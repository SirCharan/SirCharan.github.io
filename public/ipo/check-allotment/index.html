4. How to check allotment in IPO?

Jan 17 2021

Checking Allotment of IPO#

To check the allotment of an IPO, there are 2 different website I know of:

  1. BSE India
  2. Link Intime India Private Ltd.

BSE India#

Click this link to head over to the IPO application check on BSE India’s official website

  1. Select Equity
  2. Choose Issue name of the IPO you applied for
  3. Enter the application number (you can get this from your broker’s website)
  4. Enter your PAN number
  5. Select Search

OR

Link Intime Pvt. Limited#

Link Intime is more user friendly and offers you 3 options on its website:

  1. PAN Number
  2. Application Number
  3. DP Client ID
